# 🛡️ Tax-Sentinel AI

### Autonomous Neuro-Symbolic Forensic Audit & Compliance Engine

🚨 **Detect. Explain. Enforce.**

Tax-Sentinel AI is a **next-generation forensic intelligence system** designed to detect financial fraud, tax evasion, and shell network activity using a hybrid **Machine Learning + Statutory Law Engine**.

Unlike traditional black-box ML tools, Tax-Sentinel provides **legal-grade explainability**, making it suitable for **auditors, investigators, and compliance authorities**.

---

## 🔥 Why This Project Stands Out

* 🧠 **Neuro-Symbolic AI** → Combines ML intelligence with legal reasoning
* ⚖️ **Explainable Verdicts** → Every flag is backed by statutory sections
* 🌐 **Graph Intelligence** → Detects hidden money flow networks
* 🔐 **Forensic Integrity** → Immutable audit trail via SHA-256 hashing
* ⚡ **Real-Time Simulation** → Supports live transaction monitoring

---

## 🚀 Core Features

### 🧠 Hybrid Detection Engine

* Supervised ML (Isolation Forest / anomaly detection)
* Rule-based statutory override system
* Final verdict = ML + Law + Graph fusion

### 🕸️ Graph-Based Forensics

* NetworkX powered analysis
* PageRank + Degree Centrality
* Detects:

  * Circular Trading
  * Shell Company Networks
  * High Influence Nodes

### ⚖️ Compliance Engine

* Section **269ST** → Cash transaction violations
* Section **40A(3)** → Expense disallowance
* Auto-tagged legal reasoning for every flagged transaction

### 🔐 Forensic Integrity Layer

* SHA-256 Digital Fingerprint
* Ensures:

  * Data immutability
  * Tamper-proof audit trail

### 📊 Investigator Dashboard (SaaS UI)

* Real-time monitoring interface
* Priority risk detection
* Network visualization
* Legal evidence generation
